# Concept : Zones de dessin thématiques dans les fiches PDF

## L'idée

Un différenciateur clé de Koalio : chaque fiche PDF contient des **zones de dessin avec amorces** que l'enfant complète à la main après impression. Ce n'est pas un espace vide — c'est une structure visuelle pré-dessinée qui guide l'enfant.

**Deux axes se combinent :**
- Le **type d'exercice** dicte le FORMAT de la zone (partager en groupes, colorier des fractions, tracer...)
- Le **thème fun** habille l'UNIVERS (des oeufs de dino au lieu de billes, des coffres pirates au lieu de sacs...)

---

## Pourquoi c'est puissant

1. **Pédagogique** — Le dessin aide à la compréhension, surtout en P1-P3. L'enfant manipule visuellement avant de calculer.
2. **Engageant** — L'enfant ne fait pas "des maths", il aide un dinosaure à ranger ses oeufs.
3. **Différenciant** — Aucun concurrent ne propose ça. Les fiches génériques ont des espaces vides sans structure.
4. **Imprimable** — Ça fonctionne parfaitement en PDF imprimé, pas besoin d'écran.

---

## Types d'amorces par catégorie d'exercice

### 1. Division / Partage

L'amorce montre les objets à répartir en haut et les conteneurs vides en bas.

```
┌─────────────────────────────────────────────┐
│  🥚🥚🥚🥚🥚🥚🥚🥚🥚🥚🥚🥚🥚🥚🥚          │
│                                             │
│   ╭───────╮    ╭───────╮    ╭───────╮       │
│   │ Nid 1 │    │ Nid 2 │    │ Nid 3 │       │
│   │       │    │       │    │       │       │
│   │       │    │       │    │       │       │
│   ╰───────╯    ╰───────╯    ╰───────╯       │
│                                             │
│  Opération : 15 ÷ 3 = ___                  │
│  Réponse : Chaque nid contient ___ oeufs.   │
└─────────────────────────────────────────────┘
```

**L'enfant** : dessine/barre les oeufs en les répartissant dans les nids.

### 2. Fractions

L'amorce montre une forme pré-découpée en parts égales. L'enfant colorie le nombre de parts demandé.

```
┌─────────────────────────────────────────────┐
│  Colorie 3/4 de la carte au trésor :        │
│                                             │
│   ┌─────────┬─────────┐                    │
│   │         │         │                    │
│   │   1/4   │   1/4   │                    │
│   │         │         │                    │
│   ├─────────┼─────────┤                    │
│   │         │         │                    │
│   │   1/4   │   1/4   │                    │
│   │         │         │                    │
│   └─────────┴─────────┘                    │
└─────────────────────────────────────────────┘
```

### 3. Addition / Soustraction

Deux groupes d'objets. Pour l'addition, l'enfant dessine les objets ajoutés. Pour la soustraction, il barre ceux retirés.

```
┌─────────────────────────────────────────────┐
│  Le pirate a 8 pièces. Il en trouve 5.      │
│                                             │
│  Déjà là :  💰💰💰💰💰💰💰💰               │
│  Trouvées :  [dessine 5 pièces ici]         │
│                                             │
│  Total : 8 + 5 = ___                        │
└─────────────────────────────────────────────┘
```

### 4. Dénombrement

Collection partielle — l'enfant dessine les objets manquants pour atteindre le nombre demandé.

```
┌─────────────────────────────────────────────┐
│  Dessine des étoiles pour en avoir 12 :     │
│                                             │
│  ⭐⭐⭐⭐⭐⭐⭐  [dessine les manquantes]   │
│                                             │
│  Combien en as-tu ajouté ? ___              │
└─────────────────────────────────────────────┘
```

### 5. Comparaison

Deux barres côte à côte. L'enfant remplit chaque barre avec le bon nombre d'objets pour comparer visuellement.

```
┌─────────────────────────────────────────────┐
│  Qui a le plus de cristaux ?                │
│                                             │
│  Astronaute A (14) : |░░░░░░░░░░░░░░|      │
│  Astronaute B (9)  : |░░░░░░░░░|           │
│                                             │
│  14 ___ 9  (< , > ou =)                    │
└─────────────────────────────────────────────┘
```

### 6. Problème ouvert

Scène de fond thématique avec espace libre pour dessiner la situation.

```
┌─────────────────────────────────────────────┐
│  ┌─────────────────────────────────────┐    │
│  │                                     │    │
│  │     [scène de fond : jungle]        │    │
│  │                                     │    │
│  │   Dessine la situation :            │    │
│  │                                     │    │
│  │                                     │    │
│  │                                     │    │
│  └─────────────────────────────────────┘    │
│                                             │
│  Opération : _______________                │
│  Réponse : _______________                  │
└─────────────────────────────────────────────┘
```

### 7. Français — Illustre ta phrase

Après avoir écrit une phrase ou répondu à une question, l'enfant illustre.

```
┌─────────────────────────────────────────────┐
│  Écris une phrase avec le mot "aventure" :  │
│  __________________________________________ │
│                                             │
│  Dessine ta phrase :                        │
│  ┌─────────────────────────────────────┐    │
│  │            [espace libre]           │    │
│  │         bordure thématique          │    │
│  └─────────────────────────────────────┘    │
└─────────────────────────────────────────────┘
```

## Matrice thème × objets

Les thèmes fun transforment les objets génériques de l'exercice en objets de l'univers choisi.

| Objet générique | 🦕 Dinosaures | 🏴‍☠️ Pirates | 🚀 Espace | 🐨 Animaux |
|---|---|---|---|---|
| Unité à compter | Oeufs de dino | Pièces d'or | Étoiles | Noisettes |
| Conteneur | Nid | Coffre au trésor | Fusée / cratère | Terrier |
| Part à colorier | Volcan (sections) | Carte au trésor | Planète | Ruche (alvéoles) |
| Personnage | Tyrannosaure / Tricératops | Capitaine / Perroquet | Astronaute / Alien | Koala / Renard |
| Lieu / décor | Jungle préhistorique | Ile au trésor | Station spatiale | Forêt / savane |
| Barre de comparaison | Os de dino | Sabres | Météorites | Empreintes |
| Groupe (multiplication) | Troupeau | Équipage | Équipe d'exploration | Meute / famille |

### Thèmes retenus pour le MVP

1. **🦕 Dinosaures** — Très populaire P1-P3
2. **🏴‍☠️ Pirates** — Universel, mixte filles/garçons
3. **🚀 Espace** — Populaire P3-P6
4. **🐨 Animaux** — Le thème "maison" de Koalio, toujours pertinent

On peut en ajouter d'autres plus tard (Moyen Âge, Océan, Sport, Cuisine...) mais 4 suffisent pour le MVP.

---

## Quels exercices incluent une zone de dessin ?

Pas tous. C'est pertinent surtout pour :

| Niveau | Exos avec zone de dessin | Exos sans (juste texte/calcul) |
|---|---|---|
| P1-P2 | Quasi tous (l'enfant apprend par la manipulation) | Tables de multiplication, suites à compléter |
| P3-P4 | Division, fractions, problèmes, géométrie | Calcul posé, conversions, conjugaison pure |
| P5-P6 | Problèmes complexes, fractions, géométrie | Décimaux, pourcentages, grammaire, analyse |

**Règle** : au moins 1 exercice par fiche inclut une zone de dessin, même en P6.

## Approche technique envisagée

### Option retenue : SVG templates + HTML/CSS

1. **Banque d'assets SVG** par thème : chaque thème a ses objets (oeuf, nid, pièce, coffre...) en SVG simple et léger
2. **Templates d'amorce** par type d'exercice : le layout de la zone est un template HTML/CSS (grid pour la division, sections pour les fractions...)
3. **Le prompt Claude** génère les données structurées (quantité, nb de groupes, thème) et le frontend assemble le template + les assets
4. **html2canvas** capture le tout en PDF comme le reste de la fiche

### Structure des données pour une zone de dessin

```typescript
interface DrawingZone {
  type: 'division' | 'fraction' | 'addition' | 'counting' | 'comparison' | 'open' | 'illustration';
  theme: 'dinosaurs' | 'pirates' | 'space' | 'animals';
  params: {
    total?: number;        // 15 oeufs
    groups?: number;       // 3 nids
    numerator?: number;    // 3 (pour 3/4)
    denominator?: number;  // 4
    existing?: number;     // objets déjà présents
    target?: number;       // objectif à atteindre
  };
}
```

### Intégration dans le modèle Exercise existant

```typescript
interface Exercise {
  instruction: string;
  questions: Question[];
  drawingZone?: DrawingZone;  // optionnel — pas tous les exos en ont
}
```

## Exemples concrets par thème

### 🦕 Dinosaures + Division P3
> **Consigne :** Le Tricératops a pondu 12 oeufs. Il doit les répartir dans 4 nids. Combien d'oeufs dans chaque nid ?
>
> [Amorce : 12 oeufs SVG en haut, 4 nids vides en bas]
>
> Opération : 12 ÷ 4 = ___
> Réponse : ___

### 🏴‍☠️ Pirates + Fractions P3
> **Consigne :** Le Capitaine Barbe-Rousse a trouvé une carte au trésor. Colorie 3/4 de la carte.
>
> [Amorce : carte découpée en 4 sections égales]

### 🚀 Espace + Addition P1
> **Consigne :** L'astronaute a trouvé 6 étoiles. Il en trouve encore 4. Combien en a-t-il ?
>
> [Amorce : 6 étoiles dessinées + espace pointillé pour en dessiner 4]
>
> Opération : 6 + 4 = ___

### 🐨 Animaux + Problème P2
> **Consigne :** Le koala a cueilli 15 feuilles d'eucalyptus. Il en mange 7. Combien lui en reste-t-il ?
>
> [Amorce : 15 feuilles dessinées, l'enfant barre les 7 mangées]
>
> Opération : 15 − 7 = ___
> Réponse : ___
